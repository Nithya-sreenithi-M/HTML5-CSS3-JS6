page width property:

page2 width is not inherited by the child. heading paragraphs are block level elements so they take full width. If width is inherited all inline elements would acquired page2's width.

Child acquires parents properties, this is called inheritance.
Embedded styles is used and page2 color is changes(parent), child also get the parent's color.But when padding is given to page parent element it is applied to child elements.
properties which make layout collapse(padding margin) are not inherited. properties which make multiple change easy like to change font colour of the page only colour:; in page2 makes the colour change, inheritance makes it easy. or else each and every elements colour should be changed.
