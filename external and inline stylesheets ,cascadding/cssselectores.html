body is given margin:0 auto; in universal selector removed all default margin other the (user-defined) specified onces(body margin and padding).

If all paragraphs next to h2 has to be targeted , general sibling selector is used.

Css specificity

if a is given brown, all a in body gets brown. li a is given blue, a which has li parent will get blue and other a will be in brown. ul a is given green all a in ul become green li is overwritten as it is after li a. if ul a is before li a green will be overwritten by blue. This is css specificity.
