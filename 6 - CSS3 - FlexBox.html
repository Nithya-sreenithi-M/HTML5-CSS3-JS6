Flex properties

No flex:

display: flex

3 boxes got equal size due to flex and gets resized when adjusted.

Thus it creates flexibility to child elements just by using display: flex in parent div

Main and Cross axis

Row layout(default) gets vertical cross axis and horizontal main axix. Column layout gets vice versa.

justify-content: flex-start;

Justify-content in row layout align items in main axis.

justify-content: center;

justify-content: space-between;

justify-content: space-around;

align-item: strech;

Default value, this property streched our boxes and increased height as size of container irrespective of min-height.

align-item: start;

Aligns vertically in cross axis(row layout).

align-item: center;

align-item: flex-end;

flex-warp: warp;

Items were shrinking when browser is adjusted. flex-warp: warp will make the items wrap withing the container retaining the same size.

align-content: flex-end;

Will tell warp elements should start. align-item is overridden.

Try flex-start, center, space-around, space-between.

align-content and align-item will align items in cross axis. align-content will override align-item.

flex-direction: column;

Change from row layout to column layout. Now axis also changes. justify-content will align items vertically.

align-items centers horizontally.

Thus cross axis is horizontal in column based container. Main is vertical.

flex-shrink:0;

flex-box one won't shrink when screen size reduce.

flex-grow: 1;

item-3 will grow and fill the extra space.

flex-grow: 2;

item 2 will get 2 twice the amount of empty space than item-3

item 2 is not twice(flex-grow: 2) of item 3 because we have width 200px. Remaining space is added on to the 200 px

flex-basis: 0

This will assume item 2 and 3 has 0px width. Now item 2 is twice as 3

align-self: center

Overrides the align content property and aligns itself in cross axis center.

Try flex-end, flex-start. align-self also aligns items in cross axis.
