id selector

Target a single element with a specific ID attribute. Start with a hash (#) followed by the ID name. For example, to select the element with the ID "header", use #header.

class selectors

-->

Target elements with a specific class attribute. Start with a dot (.) followed by the class name. For example, to select elements with the class "button", .button.

Combined selectors

I am combining ID and class. div.class-selector - selects div with class "class-selector" gets bg color violet.

div.class-name-1.class-name-2 (1 and 2)- selects div with class "class-name-1 class-name-2" gets bg color black. (. is AND)

div.class-name-3, div.class-name-1(1 or 2)- selects div with class-name-3 and Div with class-name-1 both gets yellow. (, is or)

Descendant Selectors: ul li - selects all li inside ul. (a b is all b inside a )

ul p - selects all p tags(nexted inside li) irrespective of where it is nexted with ul scope.(all child , grand child, great grand child, etc.,)

ol .red - selects all elements inside ol which has class "red".

Child selector: span > p - selects direct childs p elements of span (no grand child p, great grand child p, etc.,). > - selects all direct childs

General Sibling Selectors: #siblings-selector ~ p - selects all p elements that are siblings of an #siblings-selector id element.Target elements that share the same parent and appear after a specific element( #siblings-selector id element (will not get selected)).

Adjacent Sibling Selectors: #siblings-selector + p - selects one direct sibling p that is immediately preceded by a specific element. If no adj p tags, then select none.

#siblings-selector + p.classname = selects none because #siblings-selector id element don't have adjacent sibling p with class classname

Universal Selector:

* targets all elements on the page.
