font size

  • Absolute unit: pixels
  • Percentage
    1. this the fault
  • Relative units:Ems
  • Relative units: REMs
  • viewport height
  • Angle Units:
    deg (Degrees): Used to specify angles, such as for rotation transforms.
    rad (Radians): Another angle unit, commonly used in trigonometric calculations.
    grad (Gradians): Less common angle unit, dividing a right angle into 100 gradians.
  • Duration Units:
    s (Seconds): Used in animations and transitions to specify durations.
    ms (Milliseconds): A smaller unit of time often used in animations and JavaScript functions.
  • Other Units:
    fr: A unit used in CSS Grid Layout and Flexbox to distribute available space proportionally.
