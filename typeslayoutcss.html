The are two types of layout. fixed width(width is given in pixel. whatever the screen size the width takes specified pixels) and liquid width( width is given in percentage(90%). whatever the screen size is 90% is taken a width) layout.

In liquid layout when the screen is minimized (when float is used)(floatelements example) the content get collapsed. so a min-width has to be specified. So that when a small size device it wont collapse.
